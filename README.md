# student-academic-success-model 📚
This project uses Logistic Regression to predict student performance based on:

📚 Study Hours

🏫 Attendance

📝 Past Scores

😴 Sleep Hours

The goal is to build a classification model that predicts whether a student will Pass or Fail.

🧠 Machine Learning Workflow

✔ Data Loading
✔ Data Preprocessing
✔ Label Encoding
✔ Feature Scaling (StandardScaler)
✔ Train-Test Split
✔ Logistic Regression Model
✔ Model Evaluation
✔ Confusion Matrix Visualization

🛠 Technologies Used

🐍 Python

📊 Pandas

🔢 NumPy

📈 Seaborn & Matplotlib

🤖 Scikit-learn
